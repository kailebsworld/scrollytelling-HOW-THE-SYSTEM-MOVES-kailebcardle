02

First Contact - console.log / DevTools

METAPHOR MOMENT

I can finally see what the type is doing. It is not random - it has a pattern. The console shows me the hidden instructions.

JS CONCEPT

console.log() + browser DevTools

WHAT SHIFTED

From "what is happening?" to "oh, I can see what is happening." Debugging becomes visibility, not magic.

03

Naming Things - variables (let, const)

METAPHOR MOMENT

Labeling type blocks so I can grab them again. Before, everything was anonymous. Now I can point and say "this one."

JS CONCEPT

let, const, variable assignment

WHAT SHIFTED

I stopped feeling like I was chasing ghosts. Variables gave me handles. I could reference the same thing twice.

VISUAL / ANIMATION NOTES

A word appears on screen. User hovers - it highlights. A label appears: "let headline = ..." The word becomes selectable.

04

The Switch - conditionals + classList.toggle

METAPHOR MOMENT

Rules that flip calm/chaos modes. "If the vibe is X, do Y." One click changes the entire visual state.

JS CONCEPT

if/else statements + classList.toggle()

WHAT SHIFTED

I realized I could design two states and flip between them. Control is not stopping motion - it is choreographing it.

05

The Memory - localStorage

METAPHOR MOMENT

The piece remembers yesterday's state. It does not reset every time. The chaos (or calm) persists.

JS CONCEPT

localStorage.setItem() / .getItem()

WHAT SHIFTED

The page stopped feeling like it forgot me. It became a space I could return to. Memory = continuity = relationship.

Closing - "I do not control it fully, but I can choreograph it."
